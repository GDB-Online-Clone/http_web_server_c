## Http Web Server Library written in C

이 라이브러리는 Http 프로토콜을 지원하는 http web server 입니다.   
   
이 라이브러리는 다음과 같은 특징을 가집니다.
- http/1.1 스펙에서 설계되었습니다.
- 웹서버 ↔ 라우터 구조를 가집니다.
- http 프로토콜만 지원합니다.
